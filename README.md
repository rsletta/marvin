# Marvin - The friendly Discord bot

.env
```
TOKEN=<discord app token>
```
## Dependencies
PM2 - Use as process manager, to keep Marvin online
```bash
npm install --global pm2
````

## Run with PM2
```bash
pm2 start index.js --name "Marvin"
```
